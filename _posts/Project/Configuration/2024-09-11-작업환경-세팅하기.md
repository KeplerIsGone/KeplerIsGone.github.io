#### 사용 중인 것들

-   **terminal** : oh-my-zsh + powerlevel10k + kitty
-   **IDE** : astronvim
-   **Tools** : yabai + skhd, sketchybar (기본 설정에 색 변경 + 커스텀 플러그인)

#### Mission Control

1.  terminal, 나중에 tmux도 써봐야겠다. 아직 kitty가 제공하는 기본 윈도우가 편하다.
2.  Brave Browser, 써본 크로뮴 브라우저 중에서 젤 좋다. 웹 서핑 + 공부용.
3.  스케줄 관리, 캘린더 + 슬랙
4.  메신저들

#### Theme

-   kitty : 커스텀, [여기](https://rootloops.sh/)서 만들었다.
-   neovim(astrovim) : 256\_noir, vim 전용 테마인줄 알았다.
